# Per-user AI scripts

Drop `.sh` files here to extend the AI command surface for your account.
Each script needs an AI-* header — see `/usr/share/doc/ai-control/scripts.md`
or `/etc/ai-control/scripts.d/hello.sh` for an example.

Invoke via:
  ai run my-script-name
or:
  ai my-script-name some args here
